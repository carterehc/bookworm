##Getting started

* To get started, clone the repo and then install gems by running 
$ bundle install

* Initialize the db
$ mysql-ctl start

* Migrate the db
$ rails db:migrate

* Run Server
$ rails s -b $IP -p $PORT

navigate to http://bookworm-[c9username].c9users.io/

* System dependencies
Ruby version: ruby 2.3.4

Rails 5.1.6

mysql2 >= 0.3.18 < 0.5
